### Step 2:

### Step 3:
 Using the np.linalg.matrix_rank(), we can find the rank of the given matrix.
### Step 4: 
## Program:
~~~ python
import numpy as np
A=np.array([[3,2,5],[1,1,2],[3,3,6]])
y=np.linalg.matrix_rank(A)
print(y)
~~~
